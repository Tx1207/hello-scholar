# 会议材料设计指南

会议材料设计的详细指南与最佳实践。

## Presentation 设计指南

### Slide 布局
- 使用 16:9 宽屏比例（现代投影仪标准）
- 保持一致页边距（建议上下左右各 1 英寸）
- 标题区域占 slide 高度的 15-20%

### Typography
- 标题字体：Sans-serif（Arial、Helvetica、Calibri）
- 正文字体：尽量与标题保持一致
- 最小字号：24pt（确保后排观众可见）
- 行距：1.2-1.5 倍

### Color Scheme
- 使用高对比度配色（深底浅字或浅底深字）
- 主色调控制在 1-2 种
- 强调色只用于突出关键信息
- 避免纯红和纯绿组合（更友好地照顾色盲用户）
- 推荐方案：
  - 学术蓝：`#003366 + #FFFFFF`
  - 专业灰：`#333333 + #F5F5F5`
  - 现代绿：`#2E7D32 + #FFFFFF`

### Visual Elements
- 图表尽量使用矢量图（SVG、PDF）
- 照片至少 300 DPI
- 图标风格统一、简洁
- 动画谨慎使用，最好不超过 3 处

## Poster 设计指南

### Layout Structure
- 标题栏占海报高度 15-20%
- 内容区划为 3-4 列，列宽尽量一致
- 结论区占海报高度 15-20%
- 页边距至少 2 英寸，便于打印和装裱

### Content Organization
- 阅读顺序遵循从左到右、从上到下
- 每个部分有清晰标题
- 用编号或箭头引导阅读路径
- 核心内容放在视觉中心附近

### Poster Typography
- 标题：72-96pt，粗体
- 作者与机构：36-48pt
- 章节标题：48-60pt
- 正文：24-32pt
- 图表标注：20-24pt

### Printing Guidelines
- 分辨率至少 150 DPI，推荐 300 DPI
- 颜色模式：打印用 CMYK，屏幕展示可用 RGB
- 文件格式：优先 PDF，其次高质量 PNG
- 打印前检查：颜色、文字清晰度、图表质量

## Promotion 内容指南

### Social Media 设计
- 配图尺寸：
  - Twitter：1200×675px
  - LinkedIn：1200×627px
  - 通用正方形：1080×1080px
- 文字叠加应保持高对比度
- Logo / 水印放在不抢占主体信息的位置

### Visual Consistency
- 统一配色方案
- 统一字体风格
- 统一图表风格
- 品牌元素（Logo、颜色）保持一致

### 最佳实践
- 简洁，不要信息过载
- 建立明确视觉层级
- 兼顾可访问性，如色盲和低视力用户
- 确保在不同平台和设备上都能正常显示
