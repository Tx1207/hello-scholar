# UI/UX Pro Max 使用说明

## 推荐顺序

1. 先运行 `--design-system`。
2. 只有首轮结果仍有明显缺口时，再补一到两个聚焦的 `--domain` 查询。
3. 写代码前先运行 `--stack`。
4. 只有当项目会跨多页或跨多个回合持续推进时，才持久化 design system。

## 好的查询示例

- `fintech analytics dashboard trust serious`
- `beauty spa wellness service elegant`
- `developer tool docs clean minimal dark`
- `education mobile app playful structured`

## 持久化 design system 规则

- `design-system/MASTER.md` 是默认权威来源。
- `design-system/pages/<page>.md` 只对对应页面覆盖主文档。
- 页面文件只记录偏差项，不要重复整份主说明。
