# 论文写作 Profile

- Profile ID: `paper-writing`
- Role: organize paper structure, write method/experiment/result sections, improve academic expression, and connect citations.
- Primary catalog source: `catalog/profiles.json`
- Typical skills: `ml-paper-writing`, `academic-polishing`, `bilingual-academic-translation`, `paper-compression-expansion`, `figure-table-captioning`, `results-report`, `citation-verification`, `writing-anti-ai`
- Typical agents: `paper-miner`

Paper claims should cite experiment packages rather than free-form memory when experiment evidence exists.

Prompt-level writing actions are handled by micro-skills: translation by `bilingual-academic-translation`, polishing by `academic-polishing`, length control by `paper-compression-expansion`, captions and diagram wording by `figure-table-captioning`, logic diagnosis by `paper-logic-diagnosis`, and result narrative by `experiment-narrative-analysis`.
