# MAP 2024：学生数学误解分析竞赛笔记

> 最近更新：2026-01-23
> 来源数量：1

---

## 竞赛概况

MAP - Charting Student Math Misunderstandings 是 Eedi 类任务的延伸版本，目标是从更完整的学生回答中识别数学误解。

### 任务形式

- 输入：题目文本 + 学生回答
- 回答形式可能是文本、图像或二者混合
- 输出：Top 3 相关误解
- 评价指标：**MAP@3**

## 任务难点

1. **多模态输入**：学生答案可能不是纯文本。
2. **部分正确现象普遍**：答案里可能同时包含正确和错误步骤。
3. **推理链依赖强**：只有理解学生如何一步步想错，才能识别误解类别。
4. **标签空间大**：误解类型依旧很多，直接分类难度很高。

## 技术趋势

### 1. MiRAGE 式多阶段推理

高分方案常见流程是：

- 先检索相似题目与误解
- 再进行多阶段 reasoning
- 最后做 ensemble 融合

### 2. 合成数据

会用 LLM 生成额外的问答、误解解释或学生错误过程，用来扩充监督信号。

### 3. 蒸馏与重排序

先用更强模型产生更好的中间信号，再蒸馏到较小模型，兼顾效果与成本。

## 和 Eedi 的差异

- Eedi 更偏向“题目 + 选项”的误解识别
- MAP 更强调“学生回答本身”的分析
- MAP 在输入形式和解释深度上更复杂，也更接近真实教育场景

## 关键经验

1. **要建模回答过程，不只是答案结果。**
2. **部分正确样本不能粗暴当作纯错误。**
3. **检索、推理、重排分层处理更稳定。**
4. **多模态支持会直接影响上限。**
