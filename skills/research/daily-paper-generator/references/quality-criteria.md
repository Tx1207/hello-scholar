# 论文质量评审标准

## 评审维度与权重

| 维度 | 权重 | 说明 |
|------|------|------|
| **创新性** | 30% | 论文的新颖性和贡献程度 |
| **方法完整性** | 25% | 方法描述的完整性与可复现性 |
| **实验充分性** | 25% | 实验设计是否全面、结果是否可信 |
| **写作质量** | 10% | 表达清晰度与学术规范性 |
| **相关性与影响力** | 10% | 与领域问题的相关性和潜在影响 |

## 详细评分标准

### 1. 创新性（30%）

- **5 分**：突破性贡献
- **4 分**：显著创新
- **3 分**：方法有新意
- **2 分**：以改进为主
- **1 分**：仅增量优化

评估要点：
- 是否提出新问题或新视角？
- 是否有实质性方法创新？
- 是否突破现有方法限制？

### 2. 方法完整性（25%）

- **5 分**：完整严谨，易复现
- **4 分**：描述详细，大部分细节可复现
- **3 分**：核心方法清楚，可基本复现
- **2 分**：关键细节不足
- **1 分**：表述不清，难以判断

评估要点：
- 方法是否清晰？
- 细节是否充分？
- 是否提供代码仓库？
- 推导是否严谨？

### 3. 实验充分性（25%）

- **5 分**：多数据集、多消融、分析深入
- **4 分**：实验较充分，对比合理
- **3 分**：主干实验完整
- **2 分**：验证不足
- **1 分**：实验明显薄弱

评估要点：
- 是否在标准数据集验证？
- 是否有充分对比？
- 是否有消融实验？
- 统计显著性如何？

### 4. 写作质量（10%）

- **5 分**：表达清晰，逻辑严密
- **4 分**：整体良好
- **3 分**：基本清楚，可理解
- **2 分**：存在模糊表述
- **1 分**：表述混乱

### 5. 相关性与影响力（10%）

- **5 分**：影响广泛
- **4 分**：对领域重要
- **3 分**：相关且有意义
- **2 分**：问题较小众
- **1 分**：影响有限

## 自动评分辅助指标

人工评审前可先看：
- 摘要里是否有结果和具体数据
- 是否使用知名数据集（如 TUH EEG、BCIC IV）
- 是否提供 GitHub 链接
- 作者机构背景（可选）
- 早期引用情况（可选）

## 综合评分计算

```
总分 = 创新性×0.30 + 方法完整性×0.25 + 实验充分性×0.25 + 写作质量×0.10 + 相关性与影响力×0.10
```

## 评审流程

1. **初筛**：根据标题和摘要排除明显不相关论文
2. **全文阅读**：对候选论文做深入阅读
3. **维度打分**：按 5 个维度分别评分
4. **计算总分**：做加权汇总
5. **排序选择**：按总分排序，选出 Top 3
