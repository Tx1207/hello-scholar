# Skill 开发检查清单

## 触发质量
- frontmatter 是否使用第三人称触发描述？
- 是否包含自然的用户触发短语？
- 是否真实反映了 skill 的作用范围？

## 主文件质量
- 快速阅读时 workflow 是否清楚？
- 边界是否写明？
- 长细节是否被下沉到 references / examples，而不是堆在主文件？

## 资源完整性
- 每个被引用文件是否存在？
- examples 是否真实且相关？
- scripts 是否真的可调用？

## 环境完整性
- 当前仓库中引用的 sibling skill / agent 是否仍然存在？
- cache、日志、编辑器临时产物是否已经清理？
